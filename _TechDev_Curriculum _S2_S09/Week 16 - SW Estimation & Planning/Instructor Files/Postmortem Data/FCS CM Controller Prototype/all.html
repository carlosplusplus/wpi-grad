General Program Information

Business Area:IRCM
Program Type:New Development
Process:IPDP Type 3
Software Category:Loosely Constrained
Software Bid As:ELOC
Customer:BAE Land and Armaments


Project Description:
The FCS CM Controller Prototype program built a prototype of a counter measure controller to use for the FCS survivability program. The goal of the prototype was to demonstrate handoff capabilities between a single sensor and countermeasure.
